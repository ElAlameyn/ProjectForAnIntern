# Задание для стажера на платформу iOS

Общее описание задания

Приложение написано для iOS и представляет собой один экран со списком данных, которые загружаются в формате JSON из интернета по ссылке. Данные парсятся и отображаются в виде списка с помощью UITableViewController, отсортированные по алфавиту.
